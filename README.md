# FCC Technical Documentation Page
A responsive technical docs page built for the FreeCodeCamp Responsive Web Design certification.

# Tech Used
- HTML5
- CSS3 (Flexbox, Grid)

# What I Practiced
- Semantic sectioning with `nav`, `main`, and `section` anchors
- Responsive two-column layout that collapses cleanly on mobile
- Accessible navigation (skip links, focus states, readable contrast)
- Using code blocks and lists to organize reference-style content
