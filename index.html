HTML

HTML (HyperText Markup Language) is the standard markup language used to create and structure content on the World Wide Web. It uses a system of tags and elements to tell web browsers how to display text, images, links, and other multimedia, essentially providing the "skeleton" or structure of a web page.

Markup Language, not a Programming Language: HTML uses tags to annotate or "mark up" text, defining its structure (e.g., this is a heading, this is a paragraph). It does not have the logical or dynamic functionality of a programming language like JavaScript.

What HTML has?

  1. Tags
  2. Content
  3. Empty Elements

CSS

Cascading Style Sheets (CSS) is a style sheet language used to define the presentation, style, and layout of documents written in markup languages like HTML. It is one of the core technologies of the World Wide Web, alongside HTML and JavaScript.

Cascading: The "cascading" nature refers to a hierarchy that determines which styles are applied when multiple, conflicting rules target the same element. More specific styles generally take precedence over less specific ones, and rules defined later in the code typically override earlier ones of equal importance.

body { background-color: lightblue; font-family: sans-serif; }

Python

Python is a high-level, general-purpose programming language known for its simplicity, readability, and versatility. It uses an English-like syntax and significant indentation to define code blocks, making it an excellent language for beginners and highly popular for a wide range of applications.

In essence, Python is a powerful, beginner-friendly language that provides an efficient and elegant way to solve problems across a broad spectrum of applications.

print("Hello, World!")

Projects Python may be good for:

  • Web Dev
  • Data Science
  • AI

Java

Java is a widely-used, high-level, object-oriented programming language designed for platform independence via the "Write Once, Run Anywhere" (WORA) principle. It uses a Java Virtual Machine (JVM) to run compiled bytecode on any device. Key features include strong security, automatic memory management (garbage collection), and versatility for web, mobile (Android), and enterprise applications.

Java remains one of the most popular languages due to its reliability and large community support.

public class HelloWorld { public static void main(String[] args) { System.out.println("Hello World"); // Prints the message to the console } }

JavaScript

JavaScript is a versatile programming language that is a core technology of the World Wide Web, alongside HTML and CSS. It is primarily used to create dynamic and interactive content on websites, such as animations, interactive maps, and functional forms.

Client-Side Scripting: JavaScript runs directly in the user's web browser, which makes interactions fast as they don't require a trip to the server. It works with the Document Object Model (DOM) to dynamically change HTML and CSS content on a page.

// This is a comment in JavaScript
console.log("Hello, World!");
